# Room AI

## Overview

**Room AI** is an intelligent design assistant that inspires people to transform their living spaces. Using advanced AI technology, Room AI analyzes your room and provides personalized recommendations for:

- **Color Schemes** - Get AI-powered suggestions for the perfect color combinations that match your style and preferences
- **Item Recommendations** - Discover furniture and decor items that complement your room design
- **Design Inspiration** - Visualize your room transformations with AI-generated suggestions

## Features

✨ **AI-Powered Design Suggestions** - Intelligent recommendations based on room analysis
🎨 **Color Palette Generation** - Curated color schemes tailored to your taste
🛋️ **Furniture & Decor Recommendations** - Smart suggestions for items that work with your design
📱 **Smooth Animations** - Beautiful animations powered by GSAP

## Tech Stack

- **Svelte** - Lightweight, reactive UI framework
- **Tailwind CSS** - Utility-first CSS framework for rapid styling
- **GSAP** - Professional animation library for smooth transitions
- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript

## Getting Started

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The application will open at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── App.svelte      # Main application component
├── app.css         # Global styles with Tailwind
└── main.js         # Application entry point
```

## How Room AI Works

1. **Room Analysis** - Input your room details and preferences
2. **AI Processing** - Our AI analyzes your space and generates recommendations
3. **Color Suggestions** - Get personalized color palettes for walls and accents
4. **Shopping Guide** - Discover recommended items with links to purchase
